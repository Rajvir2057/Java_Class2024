The following are the 15 keywords used in Java; these are reserved words with hidden meanings
1. if
2. byte
3. boolean
4.finally
5.case ,	Used in switch statements to mark blocks of text
6. class
7. catch, Catches exceptions generated by try statements
8. continue, sends control back to the loop
9. do, starts a do- while loop
10. enum, used during enumerate
11. for, begins a for loop
12. new, creates new objects
13. package, declares a java package
14. return, brings back control to the method
15. void, specifies that the method does not return value.
